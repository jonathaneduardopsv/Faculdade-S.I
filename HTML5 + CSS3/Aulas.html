Explicações das Tags

!DOCTYPE é o formato do documento que é reconhecio como HTML
Lang é a lingua que se passa a escrita do documento
head é as formatações iniciais junto com o titulo do arquivo ou site que apareceram nos mecanismos de Busca
body é espaço separado para alocar o conteúdo/codigo do meu site ou arquivo
